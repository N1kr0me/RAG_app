# AI Assistant Frontend

## Quick Start

1. **Start the server:**
   ```bash
   python ../server.py
   ```

2. **Open your browser:**
   ```
   http://localhost:3000
   ```

3. **Start chatting!**
   - Type your question about Nikhil
   - Click "Go" or press Enter
   - The AI will respond based on the knowledge base

## Files

- `index.html` - Main AI Assistant interface
- `README.md` - This file

- Modern chat interface similar to Cursor
